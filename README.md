# Simple CRUD Dengan AJAX

Asynchronous actions PHP native.

## Installation

Cloning repo
```
git clone https://github.com/linxcodev/crud-ajax.git
```

Pindah ke dalam folder:
```
cd crud-ajax
```

Buka favorit browsermu dan jalankan file ini di server anda.

## Features
1. Buat komen baru
2. Show komen
3. Edit Komen
4. Delete Komen
